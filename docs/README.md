## 快速了解

### 精灵1号能做什么？

- 打开任何网页，帮你操作
- 登录任何网站，帮你干活
- 连接服务器，帮你运维
- 理解任何需求，自己想办法完成
- 不会的东西，自己搜索学习

### 用户需要做什么？

- 在飞书添加精灵1号机器人
- 在 Web 控制台配置授权（API Key、登录网站等）
- 然后就可以在飞书里指挥精灵干活了！

### 安全吗？

- 敏感操作必须用户确认
- 支持一键终止所有操作
- 凭证加密存储
- 用户完全可控
