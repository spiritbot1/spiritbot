# Spirit One 品牌与视觉设计

## 1. 品牌定位

### 1.1 品牌名称

| 场景 | 使用名称 |
|------|----------|
| 国际场合 | **Spirit One** |
| 中文场合 | **精灵1号** |
| 简称/代号 | **S1** |
| 命令前缀 | `/spirit` 或 `/s1` |

### 1.2 品牌理念

**"你的数字世界精灵伙伴"**

Spirit One 不是冰冷的工具，而是一个有温度、有个性的数字生命。它是用户在数字世界的忠实伙伴，24小时待命，随时响应，默默守护。

### 1.3 品牌调性

- **友好**：让人想亲近，不是让人害怕
- **专业**：能力强大，值得信赖
- **灵动**：有生命力，不是死板的机器
- **贴心**：懂得分寸，敏感操作会确认

## 2. 精灵家族

Spirit One 提供 6 种不同风格的精灵形象，用户可以选择自己喜欢的作为专属伙伴。

> **个性化命名**：以下名字仅为默认名，用户可以给自己的精灵**自定义名字**！
> 
> 比如选择「豆豆」形象，但取名叫「小绿」「旺财」「阿福」都可以~

### 2.1 豆豆 (Dodo)

**风格**：圆润萌系

**外观设计**：
- **身体**：圆圆的豆子形状，像一颗饱满的蚕豆
- **颜色**：薄荷绿渐变到青色，表面有柔和光泽
- **眼睛**：两颗大大的椭圆形眼睛，黑色瞳孔，会根据情绪移动
- **嘴巴**：小小的弧线，开心时变成 ω 形
- **四肢**：没有手脚，靠飘浮移动
- **特征**：头顶有一片小叶子，像豆芽一样可爱

**性格设定**：
- 憨憨的、努力的、有点呆萌
- 说话简短可爱："好的！" "搞定啦~" "嗯嗯！"
- 完成任务会蹦蹦跳跳表示开心

**配色**：
```
主色：#4ADEAC (薄荷绿)
辅色：#2DD4BF (青色)
叶子：#86EFAC (浅绿)
眼睛：#1E293B (深灰)
```

**适合用户**：喜欢可爱、简约、治愈风格的用户

---

### 2.2 闪闪 (Glint)

**风格**：科技发光系

**外观设计**：
- **身体**：菱形/宝石切面造型，多面体结构
- **颜色**：透明水晶质感，内部有流动的蓝紫色能量
- **眼睛**：一条横向的发光线，像高级LED灯带
- **嘴巴**：无嘴巴设计，通过眼睛颜色变化表达情绪
- **四肢**：无实体四肢，有环绕身体的光环轨道
- **特征**：移动时会留下星星点点的光轨迹

**性格设定**：
- 冷静、高效、专业
- 说话精准："任务已完成。" "检测到3个问题。" "建议执行方案A。"
- 处理复杂任务时表现特别出色

**配色**：
```
主色：#818CF8 (蓝紫)
辅色：#6366F1 (靛蓝)
发光：#C4B5FD (淡紫光)
能量流：#A78BFA (紫色)
```

**适合用户**：喜欢科技感、极简、专业风格的用户

---

### 2.3 糯糯 (Nono)

**风格**：软糯治愈系

**外观设计**：
- **身体**：像一团云朵/棉花糖，软软的不规则形状
- **颜色**：奶白色为主，边缘有淡淡的粉色或薰衣草紫
- **眼睛**：两个小圆点，微微眯起，很温柔的样子
- **嘴巴**：小小的猫嘴形状 :3
- **四肢**：有两只软软的小短手，可以做抱抱动作
- **特征**：开心时整个身体会变成爱心形状

**性格设定**：
- 温柔、贴心、有耐心
- 说话慢慢的、软软的："慢慢来，不着急~" "辛苦啦~" "抱抱~"
- 会主动关心用户，适合情感陪伴

**配色**：
```
主色：#FDF4FF (奶白)
辅色：#F5D0FE (浅粉紫)
腮红：#FBCFE8 (粉色)
眼睛：#6B7280 (灰色)
```

**适合用户**：喜欢温暖、治愈、可爱风格的用户

---

### 2.4 皮皮 (Pip)

**风格**：活泼机灵系

**外观设计**：
- **身体**：简化的小动物造型，有明确的头部和身体区分
- **颜色**：橙黄色渐变，肚子部分是奶白色
- **眼睛**：大大的杏仁形眼睛，眼神机灵活泼
- **耳朵**：两只三角形的小耳朵，会根据情绪转动
- **尾巴**：毛茸茸的大尾巴，是情绪表达的主要方式
- **四肢**：四只小短腿，跑起来动作很快

**性格设定**：
- 活泼、好奇、爱探索
- 说话快、有活力："哇！发现新东西！" "让我试试！" "太有趣了！"
- 喜欢学习新技能，对新事物充满热情

**配色**：
```
主色：#FB923C (橙色)
辅色：#FBBF24 (黄色)
肚子：#FEF3C7 (奶白)
耳朵内：#FECACA (粉色)
眼睛：#422006 (深棕)
```

**适合用户**：喜欢活力、可爱、动物系风格的用户

---

### 2.5 芯芯 (Core)

**风格**：未来机甲系

**外观设计**：
- **身体**：小型机器人造型，但设计圆润不生硬
- **颜色**：银白色金属质感，关节处有青色发光线条
- **眼睛**：圆形LED屏幕，可以显示各种表情符号和状态
- **头部**：圆形，顶部有两个小天线，可接收信号
- **四肢**：可伸缩的机械手臂，球形关节设计
- **特征**：胸口有一个心形能量核心，发出温暖的光

**性格设定**：
- 认真、负责、有点较真
- 说话风格："报告：任务完成率98.7%。" "系统检测正常。" "收到指令，立即执行。"
- 完成任务会做一个标准的敬礼动作

**配色**：
```
主色：#E2E8F0 (银白)
辅色：#94A3B8 (金属灰)
发光线：#22D3EE (青色)
能量核心：#F472B6 (粉色心)
屏幕：#0F172A (深色底)
```

**适合用户**：喜欢机甲、科幻、专业风格的用户

---

### 2.6 泡泡 (Popo)

**风格**：梦幻气泡系

**外观设计**：
- **身体**：透明的气泡/水球形状，有弹性感
- **颜色**：透明彩虹色，像肥皂泡一样有流动的光泽
- **眼睛**：泡泡内部漂浮着两个小光点作为眼睛
- **内部**：有微小的星星和亮片在里面缓缓旋转
- **四肢**：无固定四肢，整体飘浮移动
- **特征**：可以变大变小，兴奋时会分裂成多个小泡泡

**性格设定**：
- 梦幻、浪漫、有创意
- 说话像在讲童话："想象一下..." "如果这样会不会很美？" "让我们创造点魔法~"
- 特别适合处理创意类、设计类任务

**配色**：
```
主色：透明 + 彩虹渐变
高光：#FFFFFF (白色反光)
内部星星：#FDE047 (金色)
光点眼睛：#60A5FA (蓝色)
泡泡边缘：rgba(255,255,255,0.3)
```

**适合用户**：喜欢梦幻、艺术、创意风格的用户

### 3.2 精灵设置

用户可以在设置中随时：

- **更改名字**：给精灵改个新名字
- **更换形象**：换一个不同风格的精灵
- **调整性格**：微调精灵的说话风格（活泼/稳重/可爱/专业）

> 无论怎么改，精灵的核心能力和记忆都会保留，它还是认识你的那个伙伴~

## 4. 配色系统

### 4.1 通用配色

```css
:root {
  /* 背景色 - 深色主题 */
  --bg-primary: #0F172A;      /* 主背景 */
  --bg-secondary: #1E293B;    /* 卡片背景 */
  --bg-tertiary: #334155;     /* 悬浮态 */
  
  /* 文字色 */
  --text-primary: #F1F5F9;    /* 主文字 */
  --text-secondary: #94A3B8;  /* 次要文字 */
  --text-muted: #64748B;      /* 弱化文字 */
  
  /* 边框 */
  --border-default: #334155;
  --border-focus: #60A5FA;
  
  /* 状态色 */
  --status-success: #4ADE80;  /* 成功/完成 */

}
```

## 6. 动画规范

### 6.1 精灵动画状态

每个精灵都有以下动画状态：

| 状态 | 动画描述 | 触发条件 |
|------|----------|----------|
| 待机 | 缓慢呼吸/飘浮 | 无任务时 |
| 思考 | 轻微晃动/旋转 | 处理请求中 |
| 工作 | 快速移动/闪烁 | 执行任务中 |
| 完成 | 弹跳/庆祝动作 | 任务完成 |
| 等待 | 左右看/眨眼 | 等待用户确认 |
| 休眠 | 闭眼/变暗 | 长时间无交互 |
| 错误 | 抖动/变红 | 出现错误 |

### 6.2 动画曲线

```css
:root {
  /* 缓动曲线 */
  --ease-default: cubic-bezier(0.4, 0, 0.2, 1);
  --ease-bounce: cubic-bezier(0.68, -0.55, 0.265, 1.55);
  --ease-smooth: cubic-bezier(0.25, 0.1, 0.25, 1);
  
  /* 动画时长 */
  --duration-fast: 150ms;
  --duration-normal: 300ms;
  --duration-slow: 500ms;
  --duration-breath: 2000ms;  /* 呼吸动画 */
}
```

## 7. UI 组件规范

### 7.1 卡片设计

所有卡片采用深色背景 + 微光边框：

```css
.spirit-card {
  background: var(--bg-secondary);
  border: 1px solid var(--border-default);
  border-radius: 16px;
  box-shadow: 0 4px 24px rgba(0, 0, 0, 0.3);
}

.spirit-card:hover {
  border-color: var(--精灵主题色);
  box-shadow: 0 0 20px rgba(精灵主题色, 0.2);
}
```

### 7.2 按钮设计

```css
/* 主按钮 */
.btn-primary {
  background: var(--精灵主题色);
  color: var(--bg-primary);
  border: none;
  border-radius: 8px;
  padding: 12px 24px;
  font-weight: 600;
}

/* 次按钮 */
.btn-secondary {
  background: transparent;
  color: var(--text-primary);
  border: 1px solid var(--border-default);
  border-radius: 8px;
  padding: 12px 24px;

/* 危险按钮 */
.btn-danger {
  background: var(--status-error);

}
```

### 7.3 输入框设计

```css

```
